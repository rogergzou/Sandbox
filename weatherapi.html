API

Your first step is grabbing weather information from a weather API.


You've probably wondering: how does your program know what the weather is today?
It is pulling information from something called an API, or application programming interface! An API allows you to access data from other sources. Today we are using a weather API !


Ever used Google Maps for directions? That is an API too!


There are some variables in the weather API: location, weather (rainy, sunny), temperature, humidity, etc. Each variable stores according value according to your geolocation.
