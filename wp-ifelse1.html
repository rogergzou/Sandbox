If...Else...

Oftentimes, you want to perform different actions for different decisions. You can use conditional statements in your code to do this. Use if to specify a block of code to be executed, if a specified condition is true. Use else to specify a block of code to be executed, if the same condition is false

Here's the general format:

if (condition) {
    block of code to be executed if the condition is true
}
else {
    block of code to be executed if the condition is false
}
