If...Else...

if-else statements: if (conditional_X){...} else {,,,}. When conditional_X is true, the first block of code executes. When it isn't true, the second block of code executes.
Use these to take different actions dynamically.

Here's the general format:

1 if (condition) {
2     block of code to be executed if the condition is true
3 } else {
4     block of code to be executed if the condition is false
5 }
