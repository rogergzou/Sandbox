Variables

Your variable can store not just values, but objects like images as well! Once an object has been stored in a variable, the variable can be used to refer to the object and to change and access the object's properties.

We demonstrate how to create a new Image Object with the Image() method.
The Image Object has property "src" which says where the image to be displayed is (for us, the Photos folder, with name "rainy.jpg").
This variable stands in for the Image Object. To call a method using the Image Object as the input, use the variable to refer to it.

var img = new Image();
img.src = "Photos/rainy.jpg";
displayImage(img);

Var img stores the created Image Object
Img.src tells the Image Object where to find the Image to display
displayImage(img) calls the displayImage method and tells it to display the Image Object stored at variable img.
